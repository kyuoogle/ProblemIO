테이블 명세서
    
    ## 1. Users (사용자)
    
    | 논리명 | 물리명 | 타입 | Null | Key | Default | 설명 |
    | --- | --- | --- | --- | --- | --- | --- |
    | ID | id | BIGINT | N | **PK** | AI | 사용자 고유 식별자 |
    | 이메일 | email | VARCHAR(100) | N | **UQ** |  | 로그인 아이디 (중복 불가) |
    | 비밀번호 | password_hash | VARCHAR(255) | N |  |  | 비밀번호 해시값 |
    | 닉네임 | nickname | VARCHAR(50) | N | **UQ** |  | 서비스 내 표시 이름 (중복 불가) |
    | 프로필 이미지 | profile_image_url | VARCHAR(500) | Y |  |  | 프로필 이미지 경로 |
    | 상태 메시지 | status_message | VARCHAR(255) | Y |  |  | 한 줄 소개 |
    | 삭제 여부 | is_deleted | TINYINT(1) | N |  | 0 | 0: 활동, 1: 탈퇴 (소프트 삭제) |
    | 가입일 | created_at | DATETIME | N |  | NOW |  |
    | 수정일 | updated_at | DATETIME | N |  | NOW |  |
    
    ## 2. Quizzes (퀴즈)
    
    | 논리명 | 물리명 | 타입 | Null | Key | Default | 설명 |
    | --- | --- | --- | --- | --- | --- | --- |
    | ID | id | BIGINT | N | **PK** | AI | 퀴즈 고유 식별자 |

    | 썸네일 | thumbnail_url | VARCHAR(500) | N |  |  | 대표 이미지 URL |
    | 공개 여부 | is_public | TINYINT(1) | N |  | 1 | 1: 공개, 0: 비공개 |

    | 플레이 수 | play_count | INT | N | IDX | 0 | 정렬용 반정규화 데이터 |
